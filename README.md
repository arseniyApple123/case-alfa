# Глобальный анализ экономического кризиса

Интерактивный веб-инструмент для анализа экономического кризиса 195 стран мира на основе 6 ключевых макроэкономических показателей с прогнозом до 2030 года.

# Возможности

- Анализ 195 стран — полная база всех признанных государств мира
- Временной охват 2020-2030 — исторические данные и прогнозы
- 6 экономических показателей:
  - ВВП (трлн долларов)
  - ВВП на душу населения (тыс долларов)
  - Уровень безработицы (%)
  - Инфляция (%)
  - Государственный долг (% ВВП)
  - Международные резервы (млрд долларов)
- Реалистичные пороговые значения:
  - ВВП не менее 0.5 трлн долларов
  - ВВП на душу не менее 15 000 долларов
  - Безработица не более 8%
  - Инфляция не более 5%
  - Госдолг не более 70%
  - Резервы не менее 50 млрд долларов
- Цветовая индикация уровня кризиса (от 0 до 6 минусов)
- Интерактивные графики:
  - Сравнение с пороговыми значениями
  - Динамика показателей 2020-2030
- Умный поиск с автодополнением
- Понятные сообщения об ошибках с предложениями похожих стран

# Быстрый старт

1. Скачайте файл index.html
2. Откройте его в любом современном браузере
3. Начните анализ

# Технологии

- HTML5 — структура приложения
- CSS3 — стилизация и адаптивный дизайн
- JavaScript — логика и интерактивность
- Chart.js — визуализация данных

# Источники данных

Данные основаны на статистике международных организаций, включая Международный валютный фонд и Всемирный банк. Прогнозы на 2027-2030 годы построены на основе среднегодовых темпов роста 2020-2025.

# Как это работает

1. Введите название страны в поле поиска
2. Выберите год из выпадающего списка (2020-2030)
3. Нажмите кнопку Анализировать
4. Получите:
   - Детальный анализ всех 6 показателей
   - Уровень кризиса с цветовой индикацией
   - Графики сравнения с пороговыми значениями
   - Динамику показателей за 10 лет
